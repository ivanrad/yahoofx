# yahoofx

Command line tool to convert currency using Yahoo Finance API.

## Installing

Clone the repository, and copy `yahoofx` shell script to a local folder (e.g. `$HOME/bin`).

## Using

		$ yahoofx 100 USD in EUR
		77.3400

		$ yahoofx 100 eur to usd
		129.3100

		$ yahoofx 100 eur eur
		100.00
